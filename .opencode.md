# Fisher Electrical, Air and Solar (Fishereas)

A high-end, single-page placeholder application built with a "Tesla-meets-John Pawson" design philosophy. Designed by **kakarotdevs**.

## 🛠 Project Overview
This project is a minimalist, professional landing page for Fishereas. It focuses on extreme simplicity, intentional white space, and high-end micro-interactions.

### **Core Design Philosophy**
- **Reductionist:** Only essential information is on screen.
- **Architectural:** Heavy influence from John Pawson (white space, thin lines, stark typography).
- **Tesla-Inspired:** Interface elements are hidden until needed; content *is* the interface.
- **Kakarotdevs Signature:** Custom interaction system featuring sliding gradient underlines, subtle lifts, and "Number Reveals".

## 🚀 Tech Stack
- **HTMX:** Used for lazy-loading sections (`services`, `contact`) and handling dynamic state swaps (Number Reveal) without a backend.
- **CSS3:** Custom system using CSS variables and modular alignment.
- **Typography:** 
  - `Inter` (Light, Regular, SemiBold) for professional utility.
  - `A-Space` (Light) for brand-specific "Fisher" hero typography.
- **Vanilla JS:** Minimal footprint for Intersection Observers and animation triggers.

## 📂 Directory Structure
- `/css/`: Core `styles.css` containing the entire design system and signature interactions.
- `/js/`: Localized `htmx.min.js`.
- `/public/fonts/`: Self-hosted web fonts for performance and stability.
- `/public/logos/`: Branding assets (`logo.svg`, `kakarot.png`).
- `/partials/`: Modular HTML components fetched by HTMX.

## 🧠 For AI Agents: Maintenance Guidelines
When modifying this project, adhere strictly to these conventions:

1.  **Spacing:** Maintain the "Compact" spacing system. Section margins should be exactly `40px` to `80px` max. Do not introduce "airy" defaults.
2.  **Typography:** 
    - "FISHER" must remain in `A-Space` (Weight 300).
    - Sub-headers in the hero must be justified to fit the width of "FISHER" using `text-align-last: justify`.
    - Use `Weight 300` for thinner emphasis and `Weight 600` for bold professional core.
3.  **Color Palette:**
    - Primary: `#0a0a0a` (Black)
    - Accents: Gradient from `var(--google-blue)` (#4285F4) to `var(--google-green)` (#00FFA3).
4.  **HTMX Patterns:** Always use `hx-get` with `intersect once` for new sections. Prefer partial swaps over full page changes.
5.  **Interactions:** Every link must use the `.page-link` or `.header-link` classes to ensure the signature gradient underline and hover-lift remain consistent.
6.  **AI Interaction:** Keep responses minimal and use dot points to save tokens.
